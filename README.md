# Android XR Experiments

This repository contains a set of individual demos and experiments that show the features and capabilities of [Android XR SDK](https://developer.android.com/develop/xr).

## Development Environment and Requirements
Android XR Experiment uses the Gradle build system and can be imported directly into [Android Studio](https://developer.android.com/studio).
